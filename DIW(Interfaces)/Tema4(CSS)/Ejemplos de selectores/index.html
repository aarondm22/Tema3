Practica de selectores: Crear un documento html que contenga una lista. Crear un fichero css para dar al cuerpo un fondo de color azul.
A la lista que le quite los estilos, centre el texto, poner una fuente, en negrita y de color blanco.
Que cada elemento de la lista tenga un espacio de 25 px y un color de fondo. con un margen de 3px.
Al pasar el puntero por encima cambie de colo y se desplace cada elemento de la lista y que si se pica sobre él cambie de color.
Tener en cuenta la teoria del color y elegir complementarios.
